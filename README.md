# STM32F1 flushing test

## Specs
F - general purpose
1 - first gen
0 - access line
3 - mainstream performance line
C - 48 pins
8 - 64 KB Flash
T - industrial temp range (-40C, +85C)
6 - LQFP circuit package

## Requirments
```txt
gcc-arm-none-eabi
gdb-arm-none-eabi
stlink-tools
```
